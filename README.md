# BIM Excel Viewer

A comprehensive tool for viewing and analyzing BIM (Building Information Modeling) data from Excel files.

## Features

- View BIM data from Excel spreadsheets
- Interactive visualization
- Data analysis and reporting
- Export capabilities

## Installation

```bash
git clone https://github.com/dzap123/bim-excel-viewer.git
cd bim-excel-viewer
```
